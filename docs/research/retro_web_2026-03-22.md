# 1990年代レトロWeb再現 - 最終レポート

## 1. エグゼクティブサマリー

1990年代のホームページとCGI/PHP掲示板を再現するにあたり、6体のAIが独立して特徴的な要素を洗い出した。結果、タイル敷き背景・アクセスカウンター・アニメーションGIF・テーブルレイアウト・フレーム分割・CGI掲示板の6要素が全員一致で中核として挙げられた。しかし最も重要な知見は、見た目の再現だけでは不十分であり、「素材の不統一さ」「掲示板の定型句」「キリ番文化」といった運営文化そのものを埋め込むことで初めて当時の空気感が立ち上がるという点である。技術的制約がそのまま美学となり、個人の手作り感が価値を持っていた時代のWebを、意図的な「粗さ」と「不完全さ」によって再現することが本プロジェクトの要諦となる。

## 2. 1990年代Webの特徴的要素一覧

### ビジュアル・装飾

| 要素 | 説明 |
|------|------|
| タイル敷き背景画像 | 星空、大理石、和紙、チェック柄などの小さなGIFを`<body background>`で全面リピート |
| アニメーションGIF | 工事中バリケード、点滅NEW!マーク、回転地球儀、燃える区切り線、踊る人形 |
| Webセーフカラー | 216色パレットに基づく`#FF0000` `#0000FF` `#00FFFF`等の原色系配色 |
| `<font>`タグ装飾 | MSゴシック/MS Pゴシック/Osakaをフォント・色・サイズごとに直接指定 |
| 装飾区切り線 | 虹色グラデーション、ドット柄のGIF画像による`<hr>`代替 |
| 相互リンクバナー | 88x31px / 200x40px の定型サイズ。GIF形式で自作・交換 |
| 減色GIF画像 | 256色ディスプレイ前提のディザリングノイズが残る画像群 |
| bevel効果の立体感 | Windows 95風の凸型陰影をテーブル枠線やボタンに適用 |

### レイアウト・構造

| 要素 | 説明 |
|------|------|
| `<frameset>`分割 | 左メニュー+右コンテンツの2-3カラム。独立スクロールバー、URL固定問題を伴う |
| `<table>`入れ子レイアウト | 段組みの唯一の手法。`border="0"`の不可視テーブルによる配置制御 |
| スペーサーGIF | 1x1pxの透明GIFで余白を物理的に制御 |
| 固定幅640-800px | `<center>`タグによる中央寄せ。レスポンシブの概念がない |
| Enterページ | 黒背景に「Welcome to My Homepage」+Enterリンクだけの玄関。推奨環境の記載 |

### インタラクション・動的機能

| 要素 | 説明 |
|------|------|
| CGI掲示板 | Kent Web / YY-BOARD / minibbs等のPerl CGI。名前・メール・タイトル・本文・削除キーの入力欄 |
| アクセスカウンター | デジタル数字風の画像カウンター+「あなたは○○人目の訪問者です」の定型文 |
| キリ番文化 | カウンターのキリ番を踏んだら掲示板報告するという参加型儀式 |
| チャットCGI | メタリフレッシュによる擬似リアルタイム通信。入退室メッセージの自動表示 |
| 投票CGI | HTMLテーブル+セル背景色で描画された棒グラフ。IP/Cookieによる簡易的な多重投票防止 |
| 日記CGI | tDiary等によるブログ以前の個人発信。「○月○日 晴れ」形式 |
| `<marquee>` `<blink>` | 横スクロール・点滅テキストによる更新告知 |
| MIDI BGM | `<bgsound>` / `<embed>`による同意なき自動再生。J-POPアレンジやゲーム音楽 |
| 隠しリンク | フッターの空白や句読点に埋め込まれた裏ページへの入口 |

### テキスト・言語文化

| 要素 | 説明 |
|------|------|
| 歓迎文 | 「ようこそ私のホームページへ!」。リンク巡回時代の初見訪問者向け挨拶 |
| 顔文字 | `(^_^)` `(T_T)` `m(_ _)m` が管理人の人柄を伝えるUIとして機能 |
| テキストアート・記号装飾 | `***` `>>>` `■■■` による見出し・区切り。低帯域時代の知恵 |
| 掲示板定型句 | 「初カキコです」「足跡残します」「また来ますね」「半年ROMれ」 |
| 注意書き・ネチケット | 「転載禁止」「荒らし禁止」「直リンク禁止」等の長文ルール |
| 更新履歴 | トップページに日付順で羅列。サイトの生存証明 |

### 運営・コミュニティ文化

| 要素 | 説明 |
|------|------|
| 管理人プロフィール | 本名は伏せハンドルネームで自己紹介。好きなゲーム・音楽等を羅列 |
| 相互リンク集 | 管理人同士の交流記録。リンク規定（「リンクフリーですが報告を」）付き |
| Webリング | 同趣味サイトが環状にリンクし合うナビゲーション |
| 開設日表記 | 「Since 1998/07/12」のフッター |
| 推奨環境明記 | 「IE5.0以上、800x600、MIDI対応」等のブラウザ・解像度指定 |
| 素材借用文化 | 「牛飼いとアイコンの部屋」等の素材配布サイトから借りた寄せ集め感 |

## 3. 再現プロジェクトへの提言

### 優先度A: まず押さえるべき骨格

1. **Enterページを設ける** -- 黒背景に歓迎文とEnterリンク、推奨環境の注意書き。サイト体験の入口として最も象徴的
2. **テーブルレイアウト+フレーム分割** -- Flexbox/Gridを封印し`<table>`入れ子で組む。左メニュー+右コンテンツのframeset構成
3. **タイル敷き背景+原色配色** -- Webセーフカラーの原色を多用し、星空や大理石のGIFを背景にリピート
4. **CGI掲示板の再現** -- 名前・メール・タイトル・本文・削除キーの5欄フォーム。ツリー型表示。POSTによるページ全体リロード

### 優先度B: 空気感を決定づける装飾

5. **アニメーションGIF群の配置** -- 工事中バリケード、回転地球、点滅NEW!を要所に。「サイトが生きている」演出
6. **アクセスカウンター** -- デジタル数字風の画像カウンター。5桁程度の数値を仕込み、キリ番文化への言及テキストを添える
7. **装飾区切り線** -- 虹色グラデーションGIFで各セクションを区切る
8. **システムデフォルトのフォームUI** -- CSSリセットをかけず、灰色立体ボーダーのボタン・セレクトボックスをそのまま使う

### 優先度C: 没入感を高める文化要素

9. **ダミーデータの埋め込み** -- 掲示板に過去ログ（定型句を含む書き込み）を仕込み、「かつて人がいた」空気感を出す
10. **管理人プロフィール+相互リンク集** -- ハンドルネーム、好きなもの羅列、88x31pxバナーを並べたリンクページ
11. **更新履歴+顔文字** -- トップに日付付き更新履歴。管理人コメントに顔文字を自然に散りばめる
12. **MIDI BGM** -- 自動再生+停止ボタン。ただし現代ブラウザの自動再生制限に注意

### 実装上の注意

- **「不統一さ」を意図的に作る**: 統一的なデザインシステムを適用しない。異なる素材集から借りたバラバラなテイストのアイコンを混在させることが本質的
- **「粗さ」を残す**: 行間の狭さ、画像のギザつき、ボタンサイズの不揃い、完璧に整列しないテーブルをあえて維持する
- **工事中ページを残す**: 全ページを完成させず「Coming Soon」や工事中バリケードGIFのページを意図的に残すことで「育っている途中」の感覚を出す
- **`<font>`タグ+インラインスタイル**: CSSファイルに分離せず、HTMLに直接記述する。ソースコードの考古学的な忠実さも再現の一部

## 4. 総括

6体のAIは技術的な要素（テーブルレイアウト、CGI、アニメーションGIF等）についてほぼ完全に合意しつつ、それぞれ独自の文化的洞察を持ち寄った。素材借用の寄せ集め感、掲示板の定型句、隠しリンクの探索性、ダミーデータによる「人の気配」の演出など、見た目の模倣を超えた「運営文化の再現」が繰り返し強調されている。1990年代Webの本質は技術的制約そのものではなく、その制約の中で個人が手作りし、訪問者と非同期的に交流した「居住空間としてのホームページ」という世界観にあると、6体の意見は収斂している。
