# 1990年代レトロHP・掲示板サイト ビジュアル・デザインリサーチ

## 1. レイアウト・構造

### テーブルレイアウト
- `<table border=0>` でページ全体の段組みを構築（CSSレイアウト以前の標準手法）
- 入れ子テーブルで複雑な配置を実現
- 1x1pxの透明GIF（spacer.gif）を拡大してマージン・パディング代わりに使用
- `&nbsp;` の連続でスペースを確保

### フレーム分割
- `<frameset cols="30%, *">` で左メニュー＋右本文の2分割が定番
- メニュークリックで右フレームだけ遷移する構成
- 現在の再現には `<iframe>` または CSS Grid で代用

### 典型的なページ構成
| ページ | 内容 |
|---|---|
| エンターページ | 「Welcome!」＋Enterボタンの導入画面 |
| トップページ | メインコンテンツへの案内 |
| 自己紹介（Profile） | 管理人プロフィール |
| 日記 | 日々の記録 |
| 掲示板（BBS） | 訪問者との交流 |
| リンク集 | 相互リンク先の一覧 |
| 素材配布 | 自作壁紙・アイコン等 |

- ページ名に世界観を反映させる文化（例: リンク集→「駅」、掲示板→「井戸端」、日記→「マスターのひとりごと」）

## 2. 配色・背景

### 背景壁紙
- 小さなGIF画像をタイル状に敷き詰め（`<body background="haikei.gif">`）
- 定番パターン: 星柄、チェック柄、花柄、大理石調、和風模様

### 配色傾向
| 系統 | 配色例 |
|---|---|
| 標準 | 白背景 `#ffffff` + 黒テキスト + 青リンク |
| ナチュラル | クリーム `#ffffcc` + 茶系テキスト |
| サイバー/ゲーム系 | 黒背景 + 蛍光グリーン・ピンク・イエロー |
| 女性向け | パステルピンク系 |
| 2ちゃんねる | 薄グレー `#efefef` + 黒テキスト + 青リンク + 暗紫既読リンク `#660099` |

### body属性での色指定（CSS以前の標準）
```html
<body bgcolor="#ffffff" text="#000000" link="#0000EE" vlink="#551A8B" alink="#FF0000">
```

## 3. フォント・テキスト装飾

### フォント
- `<font size=4 color="blue" face="MS Pゴシック">` でインライン指定
- 日本語: MS Pゴシック、MS ゴシック、Osaka
- 極小ピクセルフォント（6pt）も流行

### テキスト装飾
| 装飾 | 実装 | 備考 |
|---|---|---|
| スクロール文字 | `<marquee>` | IE独自。電光掲示板風 |
| 点滅文字 | `<blink>` | Netscape独自 |
| 虹色テキスト | 文字ごとに `<font color>` を変更 | 手作業で色指定 |
| 炎文字 | GIF画像 | テキストが燃えるエフェクト |
| 太字・斜体 | `<b>` `<i>` の多用 | |

## 4. ナビゲーション・UI要素

### アクセスカウンター
- 「あなたは **○○○人目** の訪問者です」が必須
- デジタル表示風の数字画像（CGIで生成）
- **キリ番文化**: 100, 1000, 10000等のキリ番を踏んだら掲示板に報告する義務。「踏み逃げ禁止」

### リンク・ボタン
- テキストリンクは下線付き青色、訪問済みは紫が標準
- 立体的なエンボス加工のボタン画像
- `a:hover {text-decoration: none}` がおしゃれとされた

### 88x31ピクセルバナー
- 相互リンク用の標準マイクロバナーサイズ
- ブラウザ推奨バナー（「Best viewed with IE 4.0」）
- ウェブリング参加バナー

### その他の定番要素
- `mailto:` リンクでメールアドレスを堂々掲載
- 右クリック禁止（JavaScriptでアラート表示）
- カーソル変更・キラキラ追従エフェクト
- 「Sorry. This homepage is Japanese only.」の定型文
- 「Win98+IE推奨」「文字サイズ中推奨」の注意書き

## 5. 掲示板（BBS）特有のデザイン

### 2ちゃんねるのHTML構造
```html
<html>
<head>
<meta http-equiv="Content-Type" content="text/html; charset=Shift_JIS">
<title>スレッドタイトル</title>
</head>
<body bgcolor=#efefef text=black link=blue alink=red vlink=#660099>

<h1 style="color:red;font-size:larger;font-weight:normal;">
【カテゴリ】スレッドタイトル</h1>

<dl class="thread">
<dt>1 ：<font color=green><b>名無しさん＠転載禁止</b></font>：
2014/03/12(水) 19:34:28.27 ID:XXXXX
<dd> コメント内容 <br><br>

<dt>2 ：<a href="mailto:sage"><b>名無しさん＠転載禁止</b></a>：
2014/03/12(水) 19:35:25.05 ID:mWgwEdiQ0
<dd> コメント内容 <br><br>
</dl>
```

### 2ちゃんねるの表示ルール
- 名前欄の色: メール未入力→**緑色**、メール入力あり→**青色リンク**
- `<dl>/<dt>/<dd>` 構造（定義リスト形式）
- レス表記: `レス番号 ：名前：日時（曜日付き HH:MM:SS.MS）ID:`
- トリップ: `名前#パスワード` → 一意のハッシュ文字列で本人確認
- スレッドタイトル `<h1>` は赤色
- 文字コード: Shift_JIS

### スレッドフロート式（あめぞう発祥）
- 新しい書き込みがあったスレッドが一覧の上に浮上
- 1998年にあめぞうで誕生→2ちゃんねるに引き継がれた

### CGI掲示板の投稿フォーム構成（KENT-WEB YY-BOARD等）
- 入力フィールド: 名前、メールアドレス、タイトル、コメント本文
- 顔文字GIFアイコン選択（`face/*.gif`）
- 削除キー（投稿者が自分の投稿を削除するためのパスワード）
- CSSファイル（`bbs.css`）＋HTMLテンプレート（`tmpl/*.html`）で管理

- **同盟バナー**: 共通の趣味を持つサイト群の緩い連帯
- **ネチケット講座**: ネットマナーの注意書きページ

## 8. HTML/CSSの技術的特徴

### HTML側
- `<font>` タグによるインラインスタイル指定が主体
- `<body>` 属性で全ページの色・背景を指定
- `<table border=0>` によるレイアウト
- `<marquee>`（IE）、`<blink>`（Netscape）の独自タグ
- `<center>` で中央配置
- `<embed>` / `<bgsound>` でMIDI再生
- `<frameset>` / `<frame>` で画面分割
- **メモ帳（Notepad）でHTMLベタ打ち**が主要な制作手法
- 文字コード: **Shift_JIS** が主流

### CSSの状況
- 1990年代後半はブラウザ間の互換性が低く、CSSはほぼ使われなかった
- スタイルはすべてHTMLタグ属性で記述
- 唯一普及したCSS: `a:hover {text-decoration: none; color: #000}` のみ
- CSS本格利用は2000年代以降

### ブラウザ戦争の影響
- Netscape Navigator vs Internet Explorer（第一次ブラウザ戦争）
- 「Best viewed with IE 4.0」バナーの設置文化
- タグの互換性問題（blinkはNetscapeのみ、marqueeはIEのみ）

## 再現時のポイント（実装メモ）

現代のHTML5/CSS3で再現する場合の技術的な対応:

| 90年代の技術 | 現代での代替 |
|---|---|
| `<frameset>` | CSS Grid / `<iframe>` |
| `<font>` タグ | CSS `font-family`, `color`, `font-size` |
| `<marquee>` | CSS `animation` + `@keyframes` |
| `<blink>` | CSS `animation: blink` |
| `<table>` レイアウト | CSS Grid/Flexbox（ただしあえてtableを使うのもアリ） |
| `<bgsound>` / `<embed>` | `<audio>` タグ（自動再生はブラウザが制限） |
| `<center>` | CSS `text-align: center` |
| `<body bgcolor>` | CSS `background-color` |
| spacer.gif | CSS `margin` / `padding` |
| Shift_JIS | UTF-8（表示上は問題なし） |
