# R-001: レス削除（コメント付き/なし/権限エラー）

## 対象シナリオ

```gherkin
Scenario: 管理者がコメント付きでレスを削除する
  Given 管理者がログイン済みである
  And スレッド "今日の雑談" にレス >>5 が存在する
  When レス >>5 の削除をコメント "個人情報を含むため削除しました" 付きで実行する
  Then レス >>5 の表示位置に "このレスは削除されました" と表示される
  And レス番号 >>5 は欠番にならず保持される
  And 独立したシステムレスが追加される:
    """
    ★システム
    個人情報を含むため削除しました
    """

Scenario: 管理者がコメントなしでレスを削除する
  Given 管理者がログイン済みである
  And スレッド "今日の雑談" にレス >>3 が存在する
  When レス >>3 の削除をコメントなしで実行する
  Then レス >>3 の表示位置に "このレスは削除されました" と表示される
  And レス番号 >>3 は欠番にならず保持される
  And 独立したシステムレスが追加される:
    """
    ★システム
    レス >>3 は管理者により削除されました
    """

Scenario: 管理者でないユーザーがレス削除を試みると権限エラーになる
  Given 管理者でないユーザーがログイン済みである
  When レス >>5 の削除を試みる
  Then 権限エラーメッセージが表示される
  And レスは削除されない
```

## 実装ファイル
- `src/lib/services/admin-service.ts` — AdminService（レス削除ロジック）
- `src/app/api/admin/posts/[postId]/route.ts` — DELETE APIルート
